# Plateforme web

Ce dossier contiendra la plateforme web statique.

Objectif prévu :

- importer ou prendre une photo ;
- afficher un aperçu de l'image ;
- lancer une prédiction côté navigateur ;
- afficher la prédiction et le score de confiance ;
- afficher un message clair si le résultat est incertain.

La classe `uncertain` ne sera pas entraînée. Elle dépendra seulement du score de confiance.
