# Données

Ce dossier contient les données du projet.

Le dataset complet ne doit pas être ajouté à GitHub.
Les images brutes doivent rester locales dans :

```text
data/raw/
```

Ce chemin est défini dans `config.yaml` avec la clé `paths.raw_data_dir`.

Les données éventuellement préparées ou transformées doivent rester locales dans :

```text
data/processed/
```

Ces deux dossiers sont ignorés par Git.
