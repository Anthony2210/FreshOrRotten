# Code source

Ce dossier contiendra les scripts Python du projet.

Organisation prévue :

- `data_loader.py` : chargement des images et des labels ;
- `split_strategy.py` : création des splits ;
- `model.py` : définition du baseline_model ;
- `train.py` : entraînement ;
- `evaluate.py` : évaluation ;
- `predict.py` : prédiction sur une image.
